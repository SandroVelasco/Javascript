# Como incorporar JS
Um breve guia sobre as formas de incorporação de código javascript na página.

## Sumarização
1. **Inline.**
2. **Internal.**
3. **External.**

## Inline
```html
<body onload="alert('teste')"></body>
```

## Internal
```html
<script>
  alert("teste");
</script>
```

## External
```html
<script src=""></script>
```
