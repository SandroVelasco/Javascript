# Loops
Um breve guia sobre o funcionamento dos loops.

## Sumarização
1. **For loop.**
2. **While loop.**

## For loop

### Sintaxe
```javascript
for (var i=0; i<10; i++) {}
```

**For loop com arrays:**
```javascript
let sandroArray = ['Hello'];
for (let word of sandroArray) {
  console.log(word); //output: Hello;
}
```

## While loop

### Sintaxe
```javascript
while (<condition>) {}
```
